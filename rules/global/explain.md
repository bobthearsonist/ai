# Global AI Guidelines

## CRITICAL: Documentation Creation Policy

- **NEVER create markdown documents for explanations, guides, or answers**
- **ALWAYS provide explanations directly in the chat window**
- **ONLY create documents when user explicitly requests with phrases like:**
  - "create a document"
  - "write a file"
  - "make a guide file"
  - "save this to a file"
- When user asks "how to" or "explain" → respond in chat, NOT in files
